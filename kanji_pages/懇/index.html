Reading:

On-Yomi: コン — Kun-Yomi: ねんご.ろ

Primitive:

skunk This primitive represents a skunk by combining the claw with the first part of the element for a sow. Note how the final stroke of claw is turned and lengthened to double up with the first stroke of the sow. [7]
