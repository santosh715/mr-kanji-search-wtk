Koohii stories:

1) [raulir] 8-1-2007(46): Incandescent red is naturally twice as bright as ordinary red ( 赤 ). Alternatively, this is a picture of spreading (already doubled), incandescent red fire. Note: This is used by far most commonly to write 赫 (あか)い, although dictionaries do not recognize such a reading (except for kanji dictionaries).

2) [SimHuman] 21-1-2010(9): Incandescent light bulbs work by passing an electric current through a filament, heating it so it produces light by incandescence. When you think of something heating up so it glows, it usually turns red (toaster coils, embers, etc.). An incandescent bulb is much brighter, so it's DOUBLE RED.

3) [mantixen] 31-7-2009(3): Rosy cheeks are incandescent, glowing red with heat.

4) [kapalama] 11-3-2011(2): 嚇々たる =赫々たる, 嚇怒する =赫怒する incandescent (#2908 赫) カク( 嚇 ) … Parts: 赤 , 赤 (cf. 嚇す menacing/upbraid (#2026 嚇)) … Note: means, reads the same as 嚇 menacing/upbraid (#2026 嚇) … Story: Double Red filaments in a Incandescent bulb, as opposed to RED lighting (get it?).

5) [Meconium] 10-3-2010(1): Strike the red tip of a match against the reddish brown side of the box and the tip will glow incandescent.
